```
char cval; int ival; unsigned int ui;
float fval; double dval; 
```

`(a)cval = a + 3`
- `a`: `char` -> `int` 
- `int` -> `char`

`(b)fval = ui - ival * 1.0`
- `ival`: `int` -> `double`
- `ui`: `ui` -> `double`
- `double` -> `float`

`(c)dval = ui + fval`
- `ui`: `unsigned` -> `float`
- `float` -> `double`

`(d)cval = ival + fval + dval`
- `ival`: `int` -> `float`
- `float` -> `double`
- `double` -> `char`
